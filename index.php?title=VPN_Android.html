VPN unter Android

Es gibt einige Apps, die eine VPN-Verbindung zur HSR unter Android ermöglichen. Einige brauchen Root ("Jailbreak"), andere nicht. Es gibt Tools, die den AnyConnect Zugang benutzen, und solche, die den StrongSwan Zugang benutzen.

Konfiguration

AnyConnect

Wie iPhone/iPad/Vpnc (vpn.hsr.ch / hsrremote:hsrremote_pwd / hsr_user:hsr_pwd)

StrongSwan

Wie StrongSwan (strongswan.hsr.ch / hsr_user:hsr_pwd / Ggf. "QuoVadis Root CA 2" wählen)

Tools

  • VPN Cilla, 5.00 Fr., via AnyConnect Nur ab Android 4.0, braucht kein Root, basiert auf "vpnc", hat Widget
  • VPNC Widget, gratis, via AnyConnect Braucht Root, basiert auf "vpnc", hat Widget
  • strongSwan VPN Client, gratis, via StrongSwan Nur ab Android 4.0, braucht kein Root, basiert auf "strongswan"
  • Cisco AnyConnect, offiziell, gratis, via AnyConnect Ab Android 4.0 ohne Root, vorher Herstellerabhängig oder mit Root, hat Widget
