Badgeauswertung

  • Go to [verrechnungsportal.hsr.ch] and download the CSV containing all your badge transactions.
  • Download the following script: [https://raw.githubusercontent.com/mpbzh/hsr-scripts/master/expense_analyzer.awk]
  • Then run this script like this: awk -f expense_analyzer.awk yourtransactions.csv
